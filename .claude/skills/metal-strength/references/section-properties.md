All values: mm, cm2, cm3, cm4. `Wel` is elastic and `Wpl` plastic section
modulus about the strong (y) axis. `It` is the torsion constant; for I-sections
it is approximate to within a few per cent. `i` is the radius of gyration --
buckling uses the smaller one, `iz`.

Steel weighs 7850 kg/m3, so mass per metre = A[cm2] x 0.785.

## Reading these

- **Bending capacity**, S235, class 1-2: `M_c,Rd [kNm] = Wpl[cm3] x 0.235`.
  An IPE300 gives 628 x 0.235 = **148 kNm**. For S355 multiply by 355/235.
- **Squash load**: `N [kN] = A[cm2] x f_y[MPa] / 10`. IPE300 S235 = 1264 kN --
  but buckling will cut that hard on anything but a stub.
- **Stiffness**: `EI [kNm2] = 210 x Iy[cm4] / 1000`. Deflection scales with
  1/I, so going IPE300 -> IPE400 (8356 -> 23130 cm4) cuts deflection by 64%.

SHS and RHS size: `list_sections("HEB")` or the CLI `sections --family HEB`.
